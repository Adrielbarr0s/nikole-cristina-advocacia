
# 🌐 Página Web - Advogada Nikole Cristina

Este projeto consiste no desenvolvimento de uma **página web institucional** para a advogada **Nikole Cristina**, com o objetivo de apresentar seus serviços jurídicos, áreas de atuação e facilitar o contato com seus clientes.

---

## 📋 Funcionalidades

- Página inicial com apresentação profissional  
- Seção sobre a carreira e formação  
- Áreas de atuação jurídica  
- Formulário de contato para clientes  
- Links para redes sociais e meios de comunicação  

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** e **SCSS** para estrutura e estilização  
- **Framework Angular**  
- **Responsividade** para dispositivos móveis
